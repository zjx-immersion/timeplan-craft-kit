## ✨ 用户操作指南

### 如何查看矩阵视图

1. 启动开发服务器：`pnpm dev`
2. 打开浏览器：`http://localhost:9085/`
3. 点击"Orion X 智能驾驶平台 2026 年度计划（完整版）"
4. 点击顶部的"矩阵"按钮
5. 查看矩阵视图：
   - 上下滚动查看所有产品线
   - 左右滚动查看所有时间节点
   - 悬浮在单元格上查看详细信息

### 如何查看Console Log

1. 按F12打开开发者工具
2. 切换到Console标签
3. 查看`[MatrixViewV3]`开头的日志
4. 验证数据统计是否正确
